# Cloudflare Worker 2 Command-R-Plus and Cohere Models

这是一个基于 Cloudflare Worker 平台的脚本，可以让你使用 CloudFlare 的 Worker 代理 Cohere 的 API。

# 免责声明

本免责声明适用于 GitHub 上的 “cohere-tunnel” 项目（以下简称“该项目”），改代码来源于网络并发布者制作

# 用途

1. Compute - Workers and Pages
2. 新建 Workers
3. 上传`_worker.js`或者编辑粘贴`_worker.js`的代码
4. 为Workers绑定个域名，国内便可直接访问
5. 完整的openai兼容断点举例：`https://api.youname.com/v1/chat/completions`
